<label>

El Elemento HTML <label> representa una etiqueta para un elemento en una interfaz de usuario.

Este puede estar asociado con un control ya sea mediante la utilizacion del atributo for, o ubicando el control dentro del elemento label. Tal control es llamado "el control etiquetado" del elemento label.



ATRIBUTOS DE <label>

  • for:
    → El ID del elemento de formulario etiquetable relacionado en el mismo documento que el elemento label.
    → En nuestro caso se observa que el id del primer input es "hombre" y ese mismo valor es el que se va a utilizar para el atriburto for en label.
    → Podríamos decir: asigrna una etiqueta para el id="XYZ".
    → El primer elemento en el documento con tal ID que coincida con el dispuesto en el atributo for será el control etiquetado para este elemento.

<input>

El elemento HTML <input> se usa para crear controles interactivos para formularios basados en la web con el fin de recibir datos del usuario.

ATRIBUTOS DE <input> 
  • id
    → Identificador único 
  • type = "radio"
    → Tipo de control a mostrar, en este caso un radio buttno. 
  • name="sexo"
    → Es el nombre para agrupar los distintos botones del formulario que pertencen a un mismo grupo,categoría, conjunto...
    → Observa en nuestro ejemplo, que los tres botones poseen este atributo identicamente.Reflejando así que cualquiera de estos inputs pertenecen a una categoría. 
  • value:
    → Valor que recibira el servidor cuando se seleccione el input → En este ejemplo el servidor recibira el string "ABD" si este input ha sido seleccionado por el ususario.
