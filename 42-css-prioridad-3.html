Prioridades en CSS

Para evitar conflictos y forzar el uso de un selector/estilo podemos usar !important en la propiedad 


      --CSS Externo
      
      ul { 
        color: purple !important;
      
      }

      --CSS Interno
      
      <style>
      ol {
        color: green !important;
      }
      </style>
      <link rel="stylesheet" href="assets/css/14-style-prioridad3.css">

      --HTML + CSS inline

      <ol style="color: blue;">
        <li>Primero</li>
        <li>Segundo</li>
        <li>Tercero</li>
      </ol>
      <br>
      <ul style="color: blue;">
        <li>Uno</li>
        <li>Dos</li>
        <li>Tres</li>
      </ul>
