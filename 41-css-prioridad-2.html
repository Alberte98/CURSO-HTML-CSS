      <style>
        p {
          color: green;
        }
      </style>
      <link rel="stylesheet" href="assets/css/13-style-prioridad2.css">
    
      --CSS externo
    
      p {
        color: red;
      }
      
      #el-parrafo{
        color: yellow;
      }
    
      --HTML + CSS inline
    
      <div>
        <p  style="color: blue;" > a) Aplicando CSS inline</p>
        <p id="el-parrafo" style="color: brown;"> b) Aplicando CSS interno + CSS por ID</p>
        <p> c) Aplicando CSS interno</p>
        <p> d) Aplicando CSS externo</p>
      </div>

Prioridades entre CSS inline, CSS interno y CSS externo

a) Aplicando CSS inline

b) Aplicando CSS inline + CSS por ID

c) Aplicando CSS interno

d) Aplicando CSS externo

Explicaciones:

1. En los casos a) y b) se aplica el estilo inline por tener maś prioridad que el CSS interno, externo o el ID

Mirar de siguiente forma: Aunque el elemento HTML posea un ID y este posea un selector en un archivo CSS, QUIERO QUE ESTE ID sea del color "x", y eso es más específico ya que aplica esilo al elemento que posee ese ID.

2. En los casos c) y d) se aplica el estilo externo porque el CSS externo fue llamado en el <head> posteriormente a la declaración del CSS interno y, al tratarse de elementos <p> en ambas hojas de estilo, aplica el caso de selectores del mismo tipo con propiedades en conflicto.
