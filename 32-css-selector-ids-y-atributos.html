selector de ids y atributos

Selector de ID

En un documento HTML, los selectores de ID de CSS buscan un elemento basado en el contenido del atributo ID. El atributo ID del elemento seleccionado debe coincidir exactamente con el valor dado en el selector.

Ejemplo:

Esta es una linea de ejemplo para aplicación de selectores de ID

CODIGO:

    --CSS
    #ejemplo_id {
      font-size: xx-large;
      color: darkgrey;
    }
    
    --HTML
    <p id="ejemplo_id">Esta es una linea de ejemplo con aplicación de selectores de ID</p>
