  • Uno, con tamaño y letra de <ul> → 10px
  • DOs, con tamaño 2em, es decir, dos veces el tamaño del contenedor, en este caso <ul>
  • TREs, con tamaño 2rem , es decir, dos veces el tamaño del elemento raíz, en este caso es <html>

    Observa que el tamaño SI es el doble del tamaño del elemento raíz

Todos los elementos relativos mostrados con este estilo, son relativos a las etiqueta HTML y consecuentemente con el tamaño de letra predeterminado y con el tamaño de letra del contenedor raiz
