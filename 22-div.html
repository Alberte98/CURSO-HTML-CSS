<div> en html

El elemento div es un contenedor genérico.

Como contenedor "puro" el elemento <div> no posee un significado inherente.

Dicho de otra forma. No tiene nigún efecto sobre el contenido o el diseño hasta que se le aplica un estilo de alguna manera usando css (por ejemplo, se le aplica un estilo directamente o se aplica algún tipo de modelo de diseño como Flexbox a su elemento principal)

En cambio se utiliza para:

  • Agrupar contenido de modo que se le pueda aplicar estilo fácilmente usando atributos class e id.
  • Marcando una sección de un documento como escrita en un idioma (utilizando el (langatributo)
  • etc.
