Selectores de ID vs clase

Al ser el selector de ID más espectífico que el selector de clase, se aplicará el selector de ID

NOTA:

Es posible que se defina un selector de clase y un selector de ID con el mismo nombre.

No es muy recomendable seguir esta práctica!

Selector de ID vs selector de CLASS vs selector de ATTR

Se aplicará el estilo del ID por su prioridad sobre la clase y el atributo

google 
      --CSS
      #google {
        color: crimson;
      }
    
      .texto-azul {
        color: blue;
      }
    
      [target="_blank"] {
        color: violet;
      }
      --HTML
      <div>
        <a id="google" class="texto-azul" href="http://google.com" target="_blank" rel="noopener noreferrer">google</a>
      </div>
