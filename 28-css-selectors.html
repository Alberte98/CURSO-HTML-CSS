Selectores CSS

Selector universal

Selecciona todos los elementos.

Opcionalmente, puede estar restringido a un espacio de nombres específico o a todos los espacios de nombres.

Sintaxis: * ns|* *|*

Ejemplo: * se aplicará a todos los elementos del documento.

  • Codigo CSS:

    * {
      font-style: italic;
    }

Selectores de tipo

Selecciona todos los elementos del tipo indicado.

Sintaxis: eltname

Ejemplo: h2 se aplicará a todos los elementos del documento.

  • Codigo CSS:

    h2 {
      font-style: normal;
      color: aquamarine;
    }

Selectores de clase

Selecciona los elemento que tienen el atributo de clase especificado.

Sintaxis: .clasename

Ejemplo: .index sseleccionará cualquier elemento que tenga la clase .index
